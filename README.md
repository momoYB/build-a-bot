# build-a-bot

Sample Vue.js 2 App Following the Pluralsight Vue.js Fundamentals Course

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
